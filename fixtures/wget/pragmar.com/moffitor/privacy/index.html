Moffitor Privacy Statement

Information Collected

The Moffitor application neither stores, nor transmits any data (personal or otherwise).

The pragmar.com website, as opposed to the Moffitor application, tracks usage analytics using Matomos, a self hosted analytics platform. The statistics collected include pages vistited, time on page, browser version, and more.

Updates to this Privacy Statement

The developer may make changes to this privacy statement at any time. If, in the future, Moffitor were to store or transmit data, the version number for the application would be be changed, and this privacy policy updated.
