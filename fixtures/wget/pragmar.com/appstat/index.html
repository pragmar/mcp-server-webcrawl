Comparison to OS Utilities

Windows ships with two process monitors included, Task Manager (taskmgr.exe) and Process Monitor (procmon.exe). Task Manager is a staple of OS management, but not suitable for monitoring a single application over time, while Process Monitor is powerful, but complex and often tedious to configure.

appstat bridges the gap, bringing the simplicity of Task Manager to application monitoring. You don't need a plan, you just start monitoring. If anything interesting happens during the session, you have the process data available to export.
