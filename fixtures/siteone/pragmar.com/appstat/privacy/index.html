Privacy Statement

Information Collected

The appstat application neither stores, nor transmits any personal data. This includes usage statistics.

When a process log is exported by the user, it will generate a CSV log of computer hostname, time of data collection, and a variety of resource metrics.

The pragmar.com website, as opposed to the appstat application, tracks usage analytics using Matomos, a self hosted analytics platform. The statistics collected include pages vistited, time on page, browser version, and more.

Updates to this Privacy Statement

The developer may make changes to this privacy statement at any time. If, in the future, appstat were to store or transmit data, the version number for the application would be be changed, and this privacy policy updated.
